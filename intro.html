Welcome to the image searching service.

Enter /api/imagesearch/[search terms] to search

Example:

https://polar-shelf-75989.herokuapp.com/api/imagesearch/Philadelphia

Add ?display=true to display images

Example:

https://polar-shelf-75989.herokuapp.com/api/imagesearch/Philadelphia?display=true

Paginate using offset=[number]

Example:

https://polar-shelf-75989.herokuapp.com/api/imagesearch/Philadelphia?display=true&offset=20
